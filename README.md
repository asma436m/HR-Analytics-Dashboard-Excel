## Project Title
HR Analytics Dashboard — Microsoft Excel

## Overview
Built an interactive HR dashboard analyzing employee attrition, 
salary, performance and termination data.

## Tools Used
Microsoft Excel | PivotTables | VBA | Slicers | DAX

## Key Features
- KPI cards for headcount, attrition rate, average salary
- Dynamic slicers controlling the entire dashboard
- Automated layout using VBA

## Skills Demonstrated
Data Cleaning | Dashboard Design | KPI Reporting | Data Visualizatio
